# UCP Ecosystem

The open registry of [Universal Commerce Protocol (UCP)](https://ucp.dev) tools, merchants, platforms, and resources.

**This data powers [ucplist.ai](https://ucplist.ai)** — the most comprehensive UCP directory on the web.

## What's here

- `data/listings.json` — Every UCP tool, merchant, platform, and agent integration we track
- `data/categories.json` — Category definitions
- `data/schema.json` — JSON Schema for listing validation

## Why open source?

UCP is an open protocol. Its ecosystem directory should be open too. We maintain this data so that:

- **Developers** can programmatically discover UCP-enabled services
- **AI agents** can query the registry to find merchants and tools
- **The community** can submit additions and corrections via PRs
- **Other projects** can build on this data (with attribution)

## Contributing

Found a UCP tool we're missing? Submit a PR adding it to `data/listings.json`, or [submit it through ucplist.ai](https://ucplist.ai/submit).

Please follow the schema in `data/schema.json` for new entries.

## Stats

- **35** listings tracked
- **5** categories: Merchants, Platforms, Payment Handlers, Developer Tools, Agent Integrations
